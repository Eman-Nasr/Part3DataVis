Description:

This visualization represents a map of the accidents that interact with a bar chart and line chart; the bar chart represents the number of accidents in each month, while the line chart represents the number of accidents over the hours.

Interaction:

The map has an interval selection; it selects the accidents, and that reflects on both the bar and line graphs. also, The line chart has an interval selection, but that only select on the x access and that reflects on the map, and the bar and filter only accidents that happened in those hours, while the bar chart has a point selection that allows the user to select a month and that reflect on the map and line charts filter accidents that happen on those months only, user can also choose more that one nar by holding shift while selecting.
